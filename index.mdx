## What is eigi.ai?

**eigi.ai** (powered by Agent-360) is an enterprise-grade conversational AI platform that enables businesses to create, deploy, and manage intelligent voice agents. Built with modular architecture, eigi.ai integrates cutting-edge LLM, TTS (Text-to-Speech), STT (Speech-to-Text), and telephony services to deliver natural, context-aware voice conversations at scale.

Whether you're building customer support agents, sales assistants, appointment schedulers, or survey bots, eigi.ai provides everything you need to create sophisticated voice agents without writing code.

## Technology Stack

eigi.ai is built on a modern, cloud-native stack designed for performance, scalability, and reliability:

- **Frontend**: React 18+, Material-UI, Redux Toolkit
- **Backend**: FastAPI, Python 3.11+, Uvicorn ASGI server
- **AI/ML**: Pipecat framework, multi-provider AI integrations
- **Telephony**: Plivo/Twilio API, WebRTC, WebSocket streaming
- **Database**: MongoDB with ODMantic ODM
- **Observability**: LangFuse for AI pipeline monitoring
- **Infrastructure**: Docker, AWS (ECS, Lambda, S3), Cerebrium
